# Project Instructions

## Stack
Laravel/PHP project served via Valet at http://panel.test

## Development Rules
- Run `php artisan test` or `vendor/bin/pest` before suggesting any commit
- Use Laravel Boost MCP for route inspection, DB schema, Tinker, and log access
- Check `storage/logs/laravel.log` after testing for errors
- Use Playwright to browse http://panel.test and verify no 500 errors
- NEVER commit or push without explicit user approval
